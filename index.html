"세탁의 당신" 세탁 서비스 사이트 제작

2023.06~2023.07 (6인 팀프로젝트)

세탁 서비스를 운영하는 업체라고 가정하고 서비스 사이트를 만들었습니다. html, css, javascript를 활용하여 front 작업을 하였고 Jsp를 활용하여 백앤드 작업을 진행하였습니다.

이 프로젝트 전까지도 Jsp를 활용하여 프로젝트를 한 경험이있지만, 이 프로젝트를 통해 심화된 Jsp 기술들을 활용할 수 있게 되었습니다. 프로젝트 내에서 반짝 배송 페이지, 명품관 페이지 등의 프론트 부분을 UI & UX에 깊게 고민하며 디자인하였습니다. 회원가입 및 예약리스트 페이지와 예약확인 페이지를 디자인하면 JSP를 활용하여 백앤드 작업을 진행하였습니다.

AWS 클라우드를 활용하여 배포하는 과정에서 전반적인 배포과정과 AWS 사용법을 익혀서 유익했습니다. 또한 팀원들과 Git을 사용하여 협업하는 방법을 익혀 협업에서의 문제해결능력과 더 나은 방법을 배웠습니다.

"withu" 미니 게임 사이트 개발

2023.03~2023.06 (6인 팀프로젝트)

미니 게임을 포함한 간단한 게임 사이트를 만들었습니다. 누구에게나 어렵지않고 쉽게 접근할 수 있는 게임이고, 로그인, 회원가입, 게시판 같은 백앤드 작업을 추가하여 사이트를 직접 디자인하고 제작하였습니다.

처음으로 웹이 아닌 자바스크립트를 활용한 미니게임을 개발하여 의미있는 프로젝트입니다. 캐릭터부터 게임, 웹 페이지 화면을 직접 디자인하여 제작하였습니다. 프로젝트 내에서 세션을 이용한 로그인을 제작하며 세션에 대한 이해도를 향상시켰습니다. 캐릭터 소개, 개발자 소개, 메인 페이지 일부 등 UI % UX에 깊게 고민하며 제작하였습니다.

Cafe24를 통해 배포하는 좋은 경험을 할 수 있는 프로젝트였고, 팀원들의 지인들에게 실제 설문을 받아 개발하면서 놓쳤던 부분들에 대한 피드백을 받았습니다. 실제 서비스를 하기 위해서는 다양한 부분에서 더 깊게 고민하고 개발해야한다는것을 깨달았습니다.

    게시판(댓글, 검색 기능 등), 미니게임

  •   사용기술

리움 미술관 사이트 카피 제작

2023.02~2023.03 (6인 팀프로젝트)

리움 미술관이라는 실제 웹페이지를 참고하고 카피하여 제작한 사이트입니다. 팀원들과 의논하며 실제 웹사이트에서 변경하고싶은 부분이나 아쉬운 부분을 수정하였습니다.

프로젝트의 조장을 맡은 프로젝트입니다. 팀원들과 데이터베이스 공유를 위해 aws ec2를 직접 생성하여 팀원들과 공유하고 데이터베이스 공유라는 큰 문제를 해결했던 프로젝트입니다. 단순한 기능이지만 처음으로 Jsp를 활용한 백앤드 작업을 한 프로젝트입니다. 또한 팀원들과 Git을 사용하여 처음으로 협업한 프로젝트 이기도 합니다. 처음 Git을 사용하여 merge 등 협업 방법에 있어 서툴렀지만 6명의 팀원들과 고민하고 회의하면 문제를 해결해나갔습니다.

혼자 작업이 아닌 협업하는 방법의 중요함을 많이 깨달은 프로젝트입니다. 또한 본격적으로 백앤드 분야의 흥미를 느끼게 된 프로젝트입니다.

  •   주요기능

    세션을 이용한 로그인, 회원가입,
    게시판(댓글, 검색 기능 등)
